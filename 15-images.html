HTML Screen Readers

A screen reader is a program that reads the HTML code, converts the text, and allows the user to listen to the content. Screen readers are useful for people who are visually impaired or learning disabled.

Image Maps

The map tag defines an image-map. An image-map is an image with clickable areas.

The Shapes

You must define the shape of the area, and you can choose one of these values:

  • rect - defines a rectangular region
  • circle - defines a circular region
  • poly - defines a polygonal
  • default - defines the entires region
    •
    Coordinates

    Must define coordinates to be able to place the clickable area on the image - the coordinates come in pairs, one for the x-axis and one for the y-axis

    For circles, the notation goes: "x, y, radius".
